## Setup

Install apps and [dotfiles](https://github.com/s-osa/dotfiles).

### Prerequisites

- zsh
- git
- Login to App Store

### Initialize

1. `zsh -c "$(curl -s https://raw.githubusercontent.com/s-osa/setup/refs/heads/master/init.sh)"`

### Update

1. Edit `playbook.yml`
1. Run `ansible-playbook playbook.yml`
